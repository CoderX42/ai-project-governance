## `governance/`（AI 协作治理）

> 本目录用于 AI 协作治理的提案工作区，每个变更对应一个独立 artifact 目录。

- `governance/artifacts/<change-name>/`：变更提案文件夹（由 `/governance-propose` 生成）
  - `proposal.md`：变更意图、背景、目标
  - `specs/index.md`：功能需求，P0/P1/P2 优先级
  - `specs/scenarios.md`：用户故事与使用场景
  - `design.md`：技术方案、模块边界、数据流
  - `tasks.md`：分阶段任务清单，含里程碑状态（✅ 完成 / 🔄 进行中 / ⬜ 未开始）

- `integrations/cursor/skills/ai-project-governance/SKILL.md`：AI 协作 Skill，7 个 action
- `integrations/cursor/skills/ai-project-governance/checklists/`：各 action 对应检查清单
  - `intake.md`：任务接入检查清单
  - `pre-pr.md`：PR 前置检查清单
  - `merge.md`：合并检查清单
  - `requirements.md`：需求分析检查清单
  - `implementation.md`：开发阶段检查清单
  - `propose.md`：提案生成检查清单（新增）
  - `track.md`：里程碑追踪检查清单（新增）
