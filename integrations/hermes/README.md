# Hermes Integration

Hermes is typically prompt-driven. Use the generic prompt templates.

## Setup

1. Copy the prompts from `../generic/prompts.md`.
2. Paste them into Hermes' system instruction or custom prompt configuration.
3. Place the docs from `../../docs/` into your project root so Hermes can reference them.

## Recommended Prompt Order

1. **Project Onboarding** — at the start of each session
2. **Task Intake** — before any new task
3. **Pre-PR Check** — before finishing work
4. **Merge Check** — only when explicitly merging
